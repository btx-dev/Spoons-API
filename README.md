# Spoons-API

 Card Game "Spoons" with API implementation (figures only). 
 JSON format.
 Work in progress.
 Any feedback appreciated! 

Install:
1. Create a 'config_local.php' in 'lib' directory
2. Set '$DB_user' and '$DB_pass' in it
3. Import sample db
4. Start app or hit API at /spoons.php
5. Enjoy!
